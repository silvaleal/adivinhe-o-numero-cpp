# Adivinhe o número em C++

Como eu escolhi fazer algo rápido, fiz o projeto todo em um único arquivo e não me importei com a qualidade do código.

## Como testar
```
# Preparar ambiente
git clone https://github.com/silvaleal/adivinhe-o-numero-c++ desafiocpp
cd desafiocpp

# Rodar projeto
1. Abra o terminal e use: ``g++ main.cc -o main``
2. Continue no terminal depois que criar o .exe, e rode ``./main.exe``
```
